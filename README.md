# MoocFinder
> 调用助课宝接口实现网课题库查询

接口抓取自app会有查询限制，故我在生产环境放了多个AccessToken以调用

Get Start：<https://tk.southcat.net/>  

## Features
+ 查询支持：智慧树、知到、超星、尔雅、学习通
* 纯前端实现
* 题库全
* 容易挂科（大雾
